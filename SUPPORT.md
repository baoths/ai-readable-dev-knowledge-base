# Support

## Where to ask

- **Bug reports / incorrect guidance**: open a GitHub issue.
- **Request a new KB item**: use the "New KB item request" issue template.
- **Questions / discussions**: use GitHub Discussions if enabled; otherwise open an issue with a clear question.

## Expectations

This is a community-driven, best-effort project.
There is no guaranteed response time.

## Before you open an issue

- Search existing issues and KB items first.
- Include links to the relevant KB item `id` (preferred) and file path.
- Provide a minimal reproduction or concrete example when applicable.
